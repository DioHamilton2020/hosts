# 部署方式

## 手动部署

复制文件内容粘贴到hosts文件中去。

## 自动部署

### SwitchHosts

适用于多种平台的hosts订阅管理工具。

主页：[swh.app](https://swh.app/home/)

### Linux命令行

```bash
sudo sh -c 'sed -i "/# GitHub520 Host Start/Q" /etc/hosts && curl https://raw.githubusercontent.com/DioHamilton2020/hosts/main/hosts >> /etc/hosts'
```
